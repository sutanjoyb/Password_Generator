# 🔐 Password Generator

A modern and minimal **Password Generator Web App** built with **React.js** and **Tailwind CSS**.  
It allows users to generate secure passwords with customizable options and copy them instantly.

---

## 🚀 Features

- Generate strong random passwords
- Adjustable password length (6–50)
- Toggle options:
  - Include Numbers
  - Include Symbols
- One-click copy to clipboard 📋
- Clean and responsive UI
- Minimal dark theme design
- Footer with developer attribution

---

## 🛠️ Tech Stack

- **React.js** (Hooks: useState, useEffect, useCallback)
- **Tailwind CSS** (for styling)
- **Lucide React** (for icons)
